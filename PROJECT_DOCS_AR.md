# دليل مشروع محفظة الأعمال الشخصية (React Portfolio)

مرحباً بك في دليل المطور لهذا المشروع! هذا الملف يحتوي على شرح شامل لكل جوانب المشروع، الأدوات المستخدمة، هيكلية الملفات، وكيفية التعديل عليه بسهولة.

---

## 1. نظرة عامة على المشروع (Project Overview)
هذا المشروع عبارة عن موقع "بورتفوليو" (Portfolio) شخصي عصري ومحترف مبني باستخدام مكتبة **React.js**. يهدف الموقع لعرض مهاراتك، مشاريعك، وخدماتك بطريقة جذابة ومتجاوبة مع جميع الأجهزة (Responsive).

**أهم المميزات:**
- تصميم عصري (Modern UI) مع تأثيرات حركية (Animations).
- متجاوب بالكامل مع الهواتف والحواسيب (Mobile-Responsive).
- بنية أكواد نظيفة ومنظمة (Clean Code Architecture).
- سهولة تعديل البيانات والنصوص من ملف مركزي واحد.
- لا يعتمد على مكتبات خارجية ثقيلة (No Heavy Dependencies).

---

## 2. التقنيات المستخدمة (Technologies Used)
تم بناء المشروع باستخدام مجموعة من التقنيات الحديثة لضمان الأداء العالي وسهولة الصيانة:

- **React.js (v18)**: المكتبة الأساسية لبناء واجهة المستخدم.
- **CSS3 / CSS Modules**: لتنسيق الصفحات والمكونات بشكل منفصل ومنظم.
- **React Router DOM**: للتنقل بين الصفحات (إذا تم توسيع المشروع مستقبلاً).
- **React Hooks**: استخدام `useState`, `useEffect`, `useRef` لإدارة الحالة والمنطق.
- **Custom Hooks**: تم إنشاء `useScrollAnimation` لإضافة تأثيرات الظهور عند التمرير.

## 3. هيكلية المشروع (Project Structure)
هذا هو تنظيم المجلدات والملفات داخل المشروع:

```

│   │   ├── About/          # قسم "من أنا"
│   │   ├── Contact/        # قسم "اتصل بي"
│   │   ├── Footer/         # تذييل الصفحة
│   │   ├── Hero/           # الواجهة الرئيسية (أول ما يظهر)
│   │   ├── Navbar/         # شريط التنقل العلوي
│   │   ├── Pricing/        # قسم الأسعار/الخدمات
│   │   ├── Projects/       # قسم المشاريع
│   │   ├── Skills/         # قسم المهارات
│   │   └── Testimonials/   # قسم آراء العملاء
│   ├── hooks/              # الخطافات المخصصة (Custom Hooks)
│   │   └── useScrollAnimation.js # كود التحريك عند التمرير
│   ├── App.jsx             # المكون الرئيسي الذي يجمع كل الأقسام
│   ├── App.css             # تنسيقات عامة للتطبيق

│   ├── index.css           # التنسيقات العالمية (الألوان، الخطوط)
│   └── data.js             # ملف البيانات المركزي (النصوص، المشاريع، المهارات)
└── package.json            # ملف إعدادات المشروع والمكتبات
```

## 4. الملفات الرئيسية وأهميتها

### `src/data.js` (الأهم للتعديل)
هذا الملف هو **قلب المحتوى** في الموقع. بدلاً من البحث داخل مئات الأسطر من الأكواد لتغيير نص أو صورة، يمكنك فقط تعديل هذا الملف.
يحتوي على:
- `personalInfo`: اسمك، وصفك، معلومات الاتصال.
- `projects`: قائمة مشاريعك (العنوان، الوصف، الصورة، الروابط).
- `skills`: مهاراتك ونسب إتقانها.
- `pricing`: خطط الأسعار أو الخدمات التي تقدمها.
- `testimonials`: آراء العملاء.

### `src/index.css` (الألوان والخطوط)
يحتوي هذا الملف على **متغيرات الألوان (CSS Variables)** التي تتحكم في شكل الموقع بالكامل.
- `--primary`: اللون الأساسي للموقع (حالياً البنفسجي/الأزرق).
- `--secondary`: اللون الثانوي (حالياً الوردي).
- `--dark-bg`: لون الخلفية الداكن.
- `direction`: اتجاه النصوص (تم ضبطه على `ltr` للغة الإنجليزية).

### `src/hooks/useScrollAnimation.js`
هذا ملف برمجي (Hook) مسؤول عن اكتشاف متى يظهر العنصر على الشاشة أثناء التمرير، ليقوم بإضافة "Class" يجعل العنصر يظهر بحركة انسيابية (Fade In).

## 5. دليل المكونات (Components Guide)
كل قسم في الموقع مفصول في مجلد خاص يحتوي على ملفين:
1. `Name.jsx`: كود الهيكل (HTML/React).
2. `Name.css`: تنسيقات هذا القسم فقط.

- **Navbar**: يحتوي على الروابط وزر القائمة للموبايل. يتحول لونه عند التمرير للأسفل.
- **Hero**: القسم الترحيبي الكبير. يمكنك تغيير الصورة الخلفية من `Hero.css`.
- **About**: يحتوي على نافذة الكود (Code Window) الجمالية ومعلومات عنك.
- **Projects**: يعرض المشاريع في شبكة (Grid). يأخذ بياناته تلقائياً من `data.js`.
- **Contact**: يحتوي على نموذج اتصال (شكل فقط) ومعلومات التواصل.

## 6. كيفية التعديل على المشروع (How to Customize)

### أ. تغيير النصوص والبيانات (Change Content)
1. افتح ملف `src/data.js`.
2. عدل القيم داخل علامات التنصيص.
   *   مثال: لتغيير الاسم، اذهب إلى `name: "Your Name"` واكتب اسمك.
   *   مثال: لإضافة مهارة جديدة، أضف سطر جديد داخل مصفوفة `skills`: `{ name: "New Skill", level: 80 },`.

### ب. تغيير الألوان (Change Theme)
1. افتح ملف `src/index.css`.
2. اذهب إلى قسم `:root`.
3. غير أكواد الألوان (Hex Codes).
   *   مثال: لتجعل اللون الأساسي أخضر، غير `--primary: #6366f1;` إلى `--primary: #00ff00;`.

### ج. تغيير الصور (Change Images)
- **صور المشاريع**: ضع روابط الصور في ملف `data.js` في حقل `image`.
- **صورة الخلفية (Hero)**: موجودة في ملف `src/components/Hero/Hero.css` داخل كلاس `.hero-bg-overlay`. ابحث عن `url('...')` وضع رابط صورتك.

### د. التعامل مع نموذج الاتصال (Contact Form)
حالياً، النموذج في `Contact.jsx` يقوم فقط بعرض رسالة تنبيه (`alert`) عند الإرسال. لجعله يعمل فعلياً، تحتاج لربطه بخدمة مثل **EmailJS** أو **Formspree**، حيث يمكنك إضافة كود الإرسال داخل دالة `handleSubmit`.

## 7. أوامر التشغيل (Commands)

لتشغيل المشروع على جهازك أثناء التطوير:
```bash
npm start
```
سيفتح الموقع على `http://localhost:3000`.

لبناء النسخة النهائية للرفع على الاستضافة (Production Build):
```bash
npm run build
```
سينشئ مجلد `build` يحتوي على الملفات الجاهزة للنشر.
